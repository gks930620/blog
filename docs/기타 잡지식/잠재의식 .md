# 잠재의식 메모

## 핵심 내용

- 인생의 많은 부분은 자동 반응(습관)으로 작동한다.
- 초기 경험(어린 시절 환경)이 이후 사고/행동 패턴에 큰 영향을 준다.
- 잠재의식을 바꾸는 방법으로 `최면`과 `반복`이 언급된다.
- 반복은 결국 습관을 만든다.

## 문장 요약

원하는 상태를 반복적으로 인식하고 말하고 행동하면,  
그 패턴이 잠재의식에 자리 잡아 삶의 기본 반응을 바꿀 수 있다는 관점이다.

## 실천 메모

1. 바꾸고 싶은 상태를 한 문장으로 정한다.
2. 매일 같은 문장을 반복한다.
3. 문장에 맞는 작은 행동을 같이 붙인다.
4. 일정 기간 유지하며 습관으로 고정한다.
