## GitHub Pages 설정

1. `Settings > Pages` 이동
2. `Deploy from a branch` 선택
3. Branch: `main`(또는 `master`)
4. Folder: `/docs`
5. 저장 후 1~5분 대기

## URL 예시

- 메인: `https://YOUR_ID.github.io/my-md-blog/`
- notes: `https://YOUR_ID.github.io/my-md-blog/notes/`
