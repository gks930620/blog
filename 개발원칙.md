# 프로젝트 개발 원칙

이 blog 프로젝트는 글 하나하나가 어떤 개념에 대해 잘 정리한 문서임.
즉 설명용 사이트임.


- 커밋 푸쉬는 내가 요청할 때만
- index.md에 경로 추가하는건  내가 요청할 때
  (글을 올리고 바로하는게 아니라 글 여러번 수정해야되니까.
   수정 후 최종적으로 index.md에 추가해서 사용자 볼수있게)
-  설명용 md 파일 생성되는건 기본적으로 docs 폴더(또는 그 아래 하위폴더 )



이미지 위치는 docs 폴더 아래 images 폴더로
이미지 파일들은 md이름이랑 같은 폴더에 여러개 생성해서 그 안에 넣는걸로
(문서 옮기면 이미지 위치도 같이 옮기기 )
